How This Tool Works

This tool detects and removes Gamma.app watermarks in PDF and PowerPoint files, then provides a cleaned download.

Upload Flow

  • Frontend sends the selected file to POST /api/remove-watermark as FormData.
  • Backend writes to a temp file, inspects extension, and selects the processor.
  • After cleaning, it saves outputs/processed_originalname and returns JSON with stats and download_url.

Detection Logic

  • PDF: Looks for Gamma-linked elements near the bottom-right corner (beyond ~70% width/height).
  • PPTX: Analyzes slide masters/layouts; finds shapes with hyperlinks to gamma.app positioned in the corner region.

Removal Logic

  • PDF: Removes clickable images and link annotations pointing to gamma.app; returns images_removed, links_removed, total_removed.
  • PPTX: Iterates masters and layouts; deletes watermark shapes; returns watermarks_removed and layouts_cleaned.

Outputs & Stats

  • On success, a cleaned file is written to outputs/processed_filename.
  • API responds with file_type, watermarks_removed (PDF totals), layouts_processed (PPTX), and download_url when available.
  • UI shows a single success message and a centered Download button.

Security & Validation

  • Frontend restricts to .pdf and .pptx and enforces a 50 MB limit.
  • Backend validates file type, uses secure filenames, and deletes temp uploads after processing.
